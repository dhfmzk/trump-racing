# Trump Racing

A browser game inspired by the Korean streamer content format "Trump Racing".

This project recreates the core card-racing loop as a single-page web game:

- Four horses mapped to the four card suits
- Hidden checkpoint cards that trigger penalty steps
- Manual turn progression and auto-play modes
- Round betting, bankroll tracking, and session history

## Tech Stack

- React
- TypeScript
- Vite

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
